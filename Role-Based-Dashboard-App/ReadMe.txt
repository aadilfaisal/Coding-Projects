This project is a web application architected with HTML, CSS, Django, and SQLite3, incorporating user authentication and role-based access control. Upon registration and login, users can select from predefined roles: Teacher, Parent, or Student. The role-specific dashboard offers functionalities including CRUD operations for student records and transaction management, providing a streamlined interface for data manipulation and user interaction.

Further details are given in the Word Document.

The zip folder contains all the files from the parent folder.
